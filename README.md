## 📖 About This Project

This repository contains a complete collection of web development learning materials, organized into daily lessons and hands-on exercises. It serves as a practical guide for beginners and intermediate developers looking to master HTML, CSS, and JavaScript fundamentals.

The project showcases progressive learning from basic HTML structure to advanced concepts like multimedia integration, semantic HTML, CSS styling, and interactive JavaScript elements. Each day builds upon previous lessons, creating a comprehensive learning path.

**Perfect for:**
- 🎓 Web development beginners starting their coding journey
- 💼 Developers looking to refresh their front-end fundamentals
- 📚 Students following the Sigma Web Development Course
- 🔍 Anyone seeking practical, real-world web development examples

## 🚀 Installation & Usage

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- A text/code editor (VS Code recommended)
- Basic understanding of file systems

### Quick Start

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Abhi-Coder/Sigma-Web-Development-Course.git
   ```

2. **Navigate to Project Directory**
   ```bash
   cd Sigma-Web-Development-Course
   ```

3. **Open Any Lesson**
   - Navigate to any `DAY X` folder
   - Open `index.html` in your browser
   - Or use Live Server extension in VS Code

### Using Live Server (Recommended)

1. Install Live Server extension in VS Code
2. Right-click on any `index.html` file
3. Select "Open with Live Server"
4. Browser will auto-reload on file changes

### Alternative Method

```bash
# Using Python's built-in server
cd Sigma-Web-Development-Course
python -m http.server 8000

# Then open http://localhost:8000 in your browser
```
